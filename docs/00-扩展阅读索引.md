## 数学基础

### 矩阵与向量
- **出处**：01-棋盘与规则（棋盘编码）、02-神经网络（输入输出）
- **难度**：⭐⭐（高中数学）
- **前置知识**：无
- **核心概念**：
  - 向量：一列数字，比如 `[1, 0, -1, 0, 1, 0, 0, 0, -1]`
  - 矩阵：多行多列的数字表格
  - 矩阵乘法：两个矩阵相乘得到新矩阵
- **为什么重要**：神经网络的输入是向量，所有计算都是矩阵运算
- **推荐资源**：
  - [3Blue1Brown - 线性代数的本质](https://www.youtube.com/playlist?list=PLZHQObOWTQDPD3MizzM2xVFitgF8hE_ab)（视频，强烈推荐）
  - [Khan Academy - 线性代数](https://www.khanacademy.org/math/linear-algebra)

### 概率与统计
- **出处**：03-蒙特卡洛树搜索（胜率计算）、04-自我对弈与训练（MCTS 分布）
- **难度**：⭐⭐（高中数学）
- **前置知识**：无
- **核心概念**：
  - 概率：某件事发生的可能性，0 到 1 之间
  - 期望值：多次试验的平均结果
  - 大数定律：试验次数越多，结果越接近真实概率
- **为什么重要**：MCTS 靠大量随机模拟来估算胜率，本质就是概率统计
- **推荐资源**：
  - [Khan Academy - 概率与统计](https://www.khanacademy.org/math/statistics-probability)
  - 《统计学的世界》（入门教材）

### 微积分基础
- **出处**：02-神经网络（梯度下降）、04-自我对弈与训练（反向传播）
- **难度**：⭐⭐⭐（大学数学）
- **前置知识**：函数、导数
- **核心概念**：
  - 导数：函数在某一点的变化率
  - 偏导数：多元函数对某个变量的导数
  - 梯度：所有偏导数组成的向量，指向函数增长最快的方向
  - 链式法则：复合函数的求导规则
- **为什么重要**：神经网络训练靠梯度下降，需要计算损失函数对每个权重的梯度
- **推荐资源**：
  - [3Blue1Brown - 微积分的本质](https://www.youtube.com/playlist?list=PLZHQObOWTQDMsr9K-rj53DwVRMYO3t5Yr)（视频）
  - [Khan Academy - 微积分](https://www.khanacademy.org/math/calculus-1)

## 机器学习基础

### 什么是机器学习？
- **出处**：02-神经网络、04-自我对弈与训练
- **难度**：⭐（科普级）
- **前置知识**：无
- **核心概念**：
  - 传统编程：人写规则，电脑执行
  - 机器学习：人给数据，电脑自己发现规则
  - 监督学习：给电脑"标准答案"，让它学习
  - 强化学习：给电脑"奖励信号"，让它试错学习
- **为什么重要**：AlphaGo 结合了监督学习和强化学习
- **推荐资源**：
  - [吴恩达 - 机器学习课程](https://www.coursera.org/learn/machine-learning)（经典入门课）
  - 《机器学习实战》（Peter Harrington）

### 神经网络基础
- **出处**：02-神经网络
- **难度**：⭐⭐（需要矩阵基础）
- **前置知识**：矩阵与向量
- **核心概念**：
  - 神经元：接收输入，加权求和，输出结果
  - 权重：每个输入的"重要性"
  - 偏置：神经元的"基础倾向"
  - 激活函数：给网络加入非线性（ReLU、Sigmoid、Tanh）
  - 前向传播：从输入到输出的计算过程
- **为什么重要**：这是理解 AlphaGo 神经网络的基础
- **推荐资源**：
  - [3Blue1Brown - 神经网络](https://www.youtube.com/playlist?list=PLZHQObOWTQDNU6R1_67000Dx_ZCJB-3pi)（视频，强烈推荐）
  - [Michael Nielsen - 神经网络与深度学习](http://neuralnetworksanddeeplearning.com/)（免费在线书）

### 损失函数
- **出处**：02-神经网络、04-自我对弈与训练
- **难度**：⭐⭐
- **前置知识**：神经网络基础
- **核心概念**：
  - 损失函数：衡量预测和真实答案的差距
  - 均方误差（MSE）：预测值和真实值差的平方
  - 交叉熵损失：衡量两个概率分布的差异
  - 我们的损失 = 策略损失（交叉熵）+ 价值损失（均方误差）
- **为什么重要**：训练的目标就是让损失越来越小
- **推荐资源**：
  - [PyTorch 官方文档 - Loss Functions](https://pytorch.org/docs/stable/nn.html#loss-functions)

### 梯度下降与反向传播
- **出处**：02-神经网络、04-自我对弈与训练
- **难度**：⭐⭐⭐（需要微积分基础）
- **前置知识**：微积分基础、神经网络基础
- **核心概念**：
  - 梯度下降：沿着损失下降最快的方向更新权重
  - 学习率：每次更新的步长大小
  - 反向传播：用链式法则计算每个权重的梯度
  - 批量梯度下降：用一批数据计算平均梯度
  - 随机梯度下降（SGD）：每次用一个数据计算梯度
- **为什么重要**：这是神经网络训练的核心算法
- **推荐资源**：
  - [3Blue1Brown - 反向传播](https://www.youtube.com/watch?v=Ilg3gGewQ5U)（视频）
  - [Michael Nielsen - 反向传播章节](http://neuralnetworksanddeeplearning.com/chap2.html)

### 过拟合与正则化
- **出处**：02-神经网络（扩展阅读提到）
- **难度**：⭐⭐
- **前置知识**：神经网络基础
- **核心概念**：
  - 过拟合：网络"死记硬背"训练数据，对新数据表现差
  - 欠拟合：网络太简单，连训练数据都学不好
  - 正则化：防止过拟合的技术（L1/L2 正则、Dropout）
  - 训练集 vs 测试集：用测试集检验网络是否过拟合
- **为什么重要**：井字棋状态少，网络容易过拟合；围棋状态多，需要正则化
- **推荐资源**：
  - [Kaggle - Overfitting and Underfitting](https://www.kaggle.com/learn/intro-to-machine-learning)

## 算法与数据结构

### 树与图
- **出处**：03-蒙特卡洛树搜索
- **难度**：⭐⭐（高中信息学）
- **前置知识**：无
- **核心概念**：
  - 树：一种层级结构，有根节点、子节点、叶子节点
  - 二叉树：每个节点最多两个子节点
  - 搜索树：用于表示决策过程的树
  - 遍历：访问树中所有节点的方法（深度优先、广度优先）
- **为什么重要**：MCTS 的搜索过程就是在一棵决策树上遍历
- **推荐资源**：
  - [VisuAlgo - 树可视化](https://visualgo.net/zh)
  - 《算法导论》第 12 章

### 蒙特卡洛方法
- **出处**：03-蒙特卡洛树搜索
- **难度**：⭐⭐
- **前置知识**：概率与统计
- **核心概念**：
  - 蒙特卡洛方法：用随机试验估算数学问题
  - 经典例子：用随机投点估算圆周率 π
  - 大数定律：试验次数越多，结果越准确
  - 方差：结果的波动程度，方差越小越稳定
- **为什么重要**：MCTS 的"模拟"阶段就是蒙特卡洛方法
- **推荐资源**：
  - [Wikipedia - Monte Carlo method](https://en.wikipedia.org/wiki/Monte_Carlo_method)
  - [Nature of Code - 蒙特卡洛方法](https://natureofcode.com/book/chapter-0-introduction/)

### 多臂老虎机问题
- **出处**：03-蒙特卡洛树搜索（UCB1 公式）
- **难度**：⭐⭐⭐
- **前置知识**：概率与统计
- **核心概念**：
  - 问题：面前有多台老虎机，每台胜率不同，你怎么选？
  - 探索 vs 利用：试新机器（探索）还是玩已知最好的（利用）？
  - UCB1 公式：Upper Confidence Bound，一种平衡探索和利用的策略
  - 置信区间：对真实胜率的估计范围
- **为什么重要**：MCTS 用 UCB1 决定搜索哪个分支
- **推荐资源**：
  - [Lilian Weng - Multi-Armed Bandit](https://lilianweng.github.io/posts/2018-01-23-multi-armed-bandit/)
  - [Bandit Algorithms Book](https://tor-lattimore.com/downloads/book/book.pdf)

## 深度学习进阶

### 卷积神经网络（CNN）
- **出处**：02-神经网络（提到 AlphaGo 用 CNN）
- **难度**：⭐⭐⭐
- **前置知识**：神经网络基础
- **核心概念**：
  - 卷积：用小滤波器扫描图像，提取局部特征
  - 池化：缩小特征图，减少计算量
  - 特征图：卷积后的输出
  - 为什么 AlphaGo 用 CNN：棋盘是二维的，CNN 擅长处理空间关系
- **为什么重要**：AlphaGo 的神经网络是 CNN，我们的井字棋版本简化成了全连接网络
- **推荐资源**：
  - [CS231n - 卷积神经网络](http://cs231n.github.io/convolutional-networks/)（斯坦福课程）
  - [3Blue1Brown - CNN](https://www.youtube.com/watch?v=KuXjwB4LzSA)

### 强化学习
- **出处**：04-自我对弈与训练
- **难度**：⭐⭐⭐
- **前置知识**：机器学习基础
- **核心概念**：
  - 智能体（Agent）：做决策的主体
  - 环境（Environment）：智能体交互的世界
  - 状态（State）：环境的当前情况
  - 动作（Action）：智能体可以做的事
  - 奖励（Reward）：环境给智能体的反馈
  - 策略（Policy）：状态到动作的映射
  - 价值函数（Value Function）：预测未来的累积奖励
- **为什么重要**：AlphaGo 的训练是强化学习的经典案例
- **推荐资源**：
  - [David Silver - 强化学习课程](https://www.davidsilver.uk/teaching/)（DeepMind 创始人）
  - [OpenAI Spinning Up](https://spinningup.openai.com/)（入门指南）

### 自我对弈（Self-Play）
- **出处**：04-自我对弈与训练
- **难度**：⭐⭐⭐
- **前置知识**：强化学习基础
- **核心概念**：
  - 自我对弈：AI 自己跟自己下棋
  - 为什么有效：对手总是和自己水平相当
  - 策略迭代：每局结束后更新策略，下一局用新策略
  - AlphaGo Zero：完全靠自我对弈，不学人类棋谱
- **为什么重要**：这是 AlphaGo 超越人类的关键技术
- **推荐资源**：
  - [AlphaGo Zero 论文](https://www.nature.com/articles/nature24270)
  - [OpenAI - Self-Play](https://openai.com/blog/self-play/)
