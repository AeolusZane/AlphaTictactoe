## 项目特色

- 🎯 **完整实现**：包含 AlphaGo 的所有核心组件（神经网络、MCTS、自我对弈、策略梯度）
- 📚 **通俗教程**：4 篇面向初中生的详细教程，用举例的方式讲清每个模块
- 🎬 **可视化视频**：2 个 Remotion 动画视频，直观展示对弈过程和权重更新
- 🎮 **可交互**：训练完成后可与 AI 对弈，体验训练成果
- 📊 **详细日志**：完整的训练日志，追踪学习过程

## 核心组件

| 组件 | 说明 |
|------|------|
| **Board** | 井字棋规则（落子、判胜负） |
| **PolicyValueNet** | 神经网络（策略头 + 价值头） |
| **MCTS** | 蒙特卡洛树搜索（用神经网络指导） |
| **Self-play + Train** | 自我对弈 + 策略梯度更新权重 |

## 快速开始

### 1. 运行主程序

```bash
# 安装依赖
pip install torch numpy

# 训练并对弈
python alphago_tictactoe.py
```

程序会先进行 200 局自我对弈训练，然后自动进入人机对弈模式。

### 2. 查看训练日志

```bash
# 默认模式：控制台只显示摘要，详细日志写入 training_log.txt
python alphago_tictactoe.py

# 详细模式：控制台也打印每步对弈过程
python alphago_tictactoe.py --verbose

# 直接对弈：加载已训练好的模型
python alphago_tictactoe.py --play
```

## 训练动画视频

`training-video/` 目录包含用 [Remotion](https://www.remotion.dev/) 制作的两个训练过程可视化动画视频：

### 视频 1：对弈演示（AlphaGoTicTacToe）

展示一局完整的自我对弈过程：
- 左侧：实时棋盘状态
- 右侧：MCTS 搜索分布（每个位置的胜率预测）
- 底部：每一步的详细讲解

### 视频 2：权重更新可视化（WeightUpdate）

深入讲解神经网络如何学习：
- **认识权重**：绿色=正权重，红色=负权重，线条粗细=权重大小
- **前向传播**：数据从左到右流过网络
- **计算损失**：预测与正确答案的差距
- **反向传播**：从输出往回推，计算梯度
- **更新权重**：根据梯度调整权重
- **反复训练**：损失曲线逐步下降

### 渲染视频

```bash
cd training-video
npm install
npm start        # 预览（浏览器打开 http://localhost:3000）
npm run build    # 渲染两个视频到 out/ 目录
```

输出文件：
- `out/video.mp4` - 对弈演示
- `out/weight-update.mp4` - 权重更新可视化

## 原理

1. **策略网络**：输入棋盘 → 输出每个位置的落子概率（"棋感"）
2. **价值网络**：输入棋盘 → 输出当前胜率（"大局观"）
3. **MCTS**：用神经网络缩小搜索范围 + 评估局面
4. **自我对弈**：AI 自己跟自己下，产生训练数据
5. **策略梯度**：用对弈数据更新神经网络权重
