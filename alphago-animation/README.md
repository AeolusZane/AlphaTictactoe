# AlphaGo 井字棋训练动画

用 Remotion 制作的 AlphaGo 训练过程可视化动画。

## 动画内容

- **开场**：标题和简介
- **训练演示**：展示两局自我对弈过程
  - 棋盘状态实时更新
  - MCTS 搜索分布可视化（柱状图）
  - 每步决策的概率分布
- **统计面板**：训练进度、胜率统计、损失曲线
- **结尾**：训练完成总结

## 运行

```bash
# 启动预览（浏览器打开 http://localhost:3000）
npm start

# 渲染视频
npm run build
```

## 技术栈

- Remotion 4.0
- React 19
- TypeScript

## 参数

- 分辨率：1920x1080
- 帧率：30fps
- 时长：30秒（900帧）
